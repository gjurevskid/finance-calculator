# Financial Calculator

A multi-language financial calculator with loan, savings, and budget tools.

## Features

- Loan calculator with fixed and variable rates
- Savings projector
- Budget planner
- Multi-language support (English, Macedonian, Albanian)
- Dark mode
- Multiple currency support

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Technologies Used

- Next.js 14
- TypeScript
- Tailwind CSS
- Lucide Icons
- Vercel Analytics
